Политика конфиденциальности

Приложение «Сад» (ru.farming.app) · обновлено 23 мая 2026

Настоящая Политика конфиденциальности описывает, как мобильное приложение «Сад» (идентификатор ru.farming.app) обрабатывает данные пользователя.

1. Кто мы

Разработчик приложения «Сад». По вопросам обработки данных: supportgardenrussia@gmail.com.

2. Какие данные обрабатываются

  • Данные сада — название участка, площадь, адрес, координаты, тип почвы, цели, сведения о грядках, теплицах, растениях и задачах. Хранятся локально на устройстве (база данных приложения).
  • Геолокация — запрашивается только в момент создания садового участка, чтобы определить координаты для прогноза погоды и климатических рекомендаций. Фоновый доступ к геолокации не используется.
  • Сетевые запросы — для загрузки справочника растений, прогноза погоды и обновлений приложения (при наличии подключения к интернету).
  • Диагностика — при сбоях приложение может отправлять анонимные отчёты об ошибках через Firebase Crashlytics (если на устройстве доступны сервисы Google).
  • Обновления — доставка исправлений кода может осуществляться через Shorebird Code Push между релизами в магазине приложений.

3. Цели обработки

  • планирование сада и задач;
  • получение прогноза погоды и рекомендаций по климату;
  • обеспечение стабильной работы и исправление ошибок;
  • доставка обновлений приложения.

4. Передача третьим лицам

Координаты сада не передаются третьим лицам в маркетинговых целях. Для работы функций приложения могут использоваться:

  • поставщики погодных данных (запрос прогноза по координатам участка);
  • Google Firebase Crashlytics — отчёты о сбоях;
  • Shorebird — доставка обновлений кода приложения.

Геолокация не используется в фоновом режиме. Формулировка для магазина приложений: «Приложение запрашивает доступ к геопозиции только в момент создания садового участка, чтобы автоматически определить его координаты для получения локального прогноза погоды и рекомендаций по посадке. Геолокация не используется в фоновом режиме и не передаётся третьим лицам.»

5. Хранение и удаление

Основные данные сада хранятся на устройстве пользователя. Удаление приложения с устройства приводит к удалению локальных данных. По запросу на supportgardenrussia@gmail.com мы ответим на вопросы об обработке данных.

6. Права пользователя

Вы можете не предоставлять геолокацию и ввести координаты или адрес вручную, ограничить доступ к местоположению в настройках ОС, а также обратиться в поддержку с вопросами о своих данных.

7. Изменения политики

Актуальная версия всегда доступна на этой странице. При существенных изменениях дата обновления будет указана в начале документа.
